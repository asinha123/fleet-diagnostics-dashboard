# Fleet diagnostics dashboard

## Prerequisites

Node.js ^14.20.0 || ^16.13.0 || ^18.10.0

npm 9+

Angular CLI (npm install -g @angular/cli@15)

 ## Backend Setup
cd be

npm install

npm run seed

npm run dev

### to truncate table

npm run truncate


### Backend runs on:

http://localhost:3000

## Swagger Setup
http://localhost:3000/api-docs

## Frontend Setup
cd fe

npm install

npm start


### Frontend runs on:

http://localhost:4200
